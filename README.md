# GDP Data Pipepline Analysis

**ETL & EDA**

This project focuses on the integration and exploration of GDP data. By synthesizing two disparate datasets, I created a unified 60-year longitudinal time series to analyze national economic growth trajectories.

## Setup

1. Data Engineering (ETL): Cleaning, filtering, and merging historical (1960) and modern (2020) records.

2. Analysis (EDA): Visualizing long-term trends and identifying growth patterns for specific countries.
